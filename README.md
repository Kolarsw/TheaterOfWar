# TheaterOfWar

A Palantir Gotham/Foundry-style interactive geospatial application for visualizing WWII logistics, troop movements, and battles.

## Tech Stack

- **Frontend:** Next.js, Mapbox GL JS, Deck.gl, Tailwind CSS, Zustand
- **Backend (future):** Databricks, PySpark, Delta Live Tables
- **Geospatial:** H3 hexagonal indexing, NATO APP-6 military symbology

## Getting Started

```bash
npm install
npm run dev
```
